mixin混入

mixin:里面的属性方法是独立的多组件调用不相互影响
vuex:里面的数据是共用的，而mixin数据在各个组件中是私有的
组件:父组件与子组件是相对独立的，而mixin是在父组件对象中添加新的数据，直接合成新的组件
优先级:如果名称相同，组件中的方法覆盖mixin中的方法，mixiny优先级别低
