什么是docker

docker 核心组件

  • 通过描述脚步 Dockerfile构建 image 镜像

  • 通过 image 镜像构建容器 （我们将应用程序运行所需要的环境，打包为镜像文件）

  • 容器 Container ，运行应用程序

  • 进行仓库，保存、上传、下载 image 镜像

docker 镜像

文件系统

UnionFS（联合文件系统）

​ 是一个分层，轻量，高性能的文件系统，支持对文件系统的修改作为一次提交，一层层提交，是docker 镜像的基础，通过分层继承，基于基础镜像制作各种应用镜像

  • 资源共享（如果多个镜像都从base镜像构建而来，那么宿主机只保存一份base镜像就可以了）
  • 每个镜像的每一层都可以被共享
  • 镜像是只读的，只有容器才是可写的（平时操作的容器也是一层，位于最顶部，下面的都是镜像层）

镜像

docker image镜像 只包含linux发行版，以及依赖的东西，很小，内核与宿主机共用
