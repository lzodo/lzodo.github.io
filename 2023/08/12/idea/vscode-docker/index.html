vscode 管理 docker

发表于2023-08-12|更新于2023-08-12|前端工具类
|阅读量:

准备

公私钥设置

生成公私钥，将公私钥配置到服务器

vscode 连接服务器

  • 安装 Remote SSH
  • vscode 右下角打开远程连接窗口
  • 选择连接到主机并打开

docker 配置

  • 连接到远程后，在远程 vscode 窗口安装 docker 插件
  • 最左边出现小鲸鱼选项
